# easier-home-digital-ecosystem
Ecosistema digital completo para asesoría inmobiliaria  de lujo en Marbella: web bilingüe ES/EN, 6 formularios  de captación automatizados con Make → Google Sheets,  Google Business Profile, TikTok, WhatsApp Business y  estrategia de contenido con IA. Stack: Make · GTM ·  Google Sheets · Gamma · IA generativa.
